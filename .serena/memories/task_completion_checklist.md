# Task Completion Checklist

## After completing a coding task:
1. **No linter/formatter configured** — manually verify code style consistency
2. **No test suite configured** — verify changes work by reading the code carefully
3. **ES module syntax** — ensure `import`/`export` (not `require`/`module.exports`)
4. **Environment variables** — never commit `.env`; it's in `.gitignore`
5. **Git** — the project does not currently have a git repository initialized
